# cpp20_module_example
examples of simple cpp20 module feature with with cmake and with g++ compiler


## USAGE

### With using bash

First, go to ```with_gcc ``` folder with

```cd with_gcc ```

then use

```./build.sh ```


### With using cmake

First, go to ``` with_cmake ``` folder with

```cd with_cmake ```

then use

```cmake . -B build ```
```make -C build```
```ninja -C build```
